# Linux DevOps Commands Practice

This repository contains commonly used Linux commands for DevOps engineers.

-----------------------------------

## File and Directory Commands

pwd
Print working directory

ls
List files and directories

cd
Change directory

mkdir foldername
Create new folder

rm filename
Remove file

rm -r foldername
Remove folder

-----------------------------------

## File Permissions

chmod 777 file.txt
Change file permissions

chown user:group file.txt
Change file ownership

-----------------------------------

## Process Monitoring

ps
Show running processes

top
Monitor system processes

htop
Advanced process monitor

-----------------------------------

## Disk Usage

df -h
Check disk usage

du -sh *
Check folder size

-----------------------------------

## Memory Usage

free -m
Show memory usage

-----------------------------------

## Networking Commands

ping google.com
Check network connectivity

curl google.com
Fetch web content

netstat -tulnp
Check open ports

-----------------------------------

## Package Management

sudo apt update
Update packages

sudo apt install docker.io
Install Docker

-----------------------------------
